# Cara Reset Password Jika Lupa

## Langkah-langkah Reset Password:

### 1. Buka Halaman Login
- Akses aplikasi di browser
- Anda akan melihat halaman login

### 2. Klik "Lupa Password?"
- Di bagian bawah form login, klik link **"Lupa Password?"**
- Halaman akan berubah ke form reset password

### 3. Masukkan Email
- Masukkan alamat email yang terdaftar di akun Anda
- Pastikan email yang dimasukkan benar dan aktif
- Klik tombol **"Kirim Link Reset Password"**

### 4. Cek Email Anda
- Buka aplikasi email (Gmail, Yahoo, Outlook, dll)
- Cari email dari Supabase dengan subject tentang reset password
- **Jika tidak ada di Inbox, cek folder Spam/Junk**

### 5. Klik Link di Email
- Buka email reset password
- Klik link yang disediakan
- Link akan membawa Anda ke halaman reset password

### 6. Masukkan Password Baru
- Masukkan password baru yang kuat
- Konfirmasi password baru
- Klik **"Reset Password"**

### 7. Login dengan Password Baru
- Setelah berhasil reset, Anda akan diarahkan ke halaman login
- Login menggunakan email dan password baru

---

## Tips Keamanan Password:

### Password yang Kuat:
- ✅ Minimal 8 karakter
- ✅ Kombinasi huruf besar dan kecil
- ✅ Mengandung angka
- ✅ Mengandung simbol (!@#$%^&*)

### Contoh Password Kuat:
- `SekolahKita2025!`
- `Laporan@123`
- `MTA_Tauhid99`

## Troubleshooting:

### Jika Email Reset Tidak Masuk:
1. **Cek Folder Spam/Junk**
   - Email reset sering masuk ke spam
   - Tandai sebagai "Not Spam" jika ditemukan

2. **Tunggu Beberapa Menit**
   - Email mungkin butuh waktu 1-5 menit untuk sampai

3. **Cek Email yang Benar**
   - Pastikan email yang dimasukkan sama dengan saat registrasi
   - Coba ingat email alternatif yang mungkin digunakan

4. **Coba Lagi**
   - Jika masih belum masuk, coba request reset lagi
   - Tunggu minimal 1 menit sebelum request ulang

### Jika Link Reset Expired:
- Link reset password biasanya berlaku 1 jam
- Jika expired, request reset password baru
- Gunakan link yang paling baru

### Jika Masih Bermasalah:
1. **Clear Browser Cache**
   - Ctrl+Shift+Delete → Clear cache
   - Restart browser

2. **Coba Browser Lain**
   - Chrome, Firefox, Edge, Safari

3. **Hubungi Admin**
   - Jika semua cara di atas gagal
   - Admin dapat reset password secara manual

## Pencegahan Lupa Password:

### Simpan Password dengan Aman:
- ✅ Gunakan Password Manager (Chrome, Firefox, 1Password, dll)
- ✅ Tulis di tempat yang aman
- ✅ Jangan gunakan password yang sama untuk semua akun

### Update Email:
- Pastikan email yang terdaftar masih aktif
- Update email jika sudah tidak digunakan

### Login Berkala:
- Login minimal seminggu sekali
- Agar tidak lupa password

---

## Fitur Reset Password Tersedia:
✅ Form "Lupa Password" di halaman login  
✅ Email reset otomatis dari sistem  
✅ Link reset yang aman dan ter-enkripsi  
✅ Validasi password baru  
✅ Redirect otomatis setelah berhasil  

Jika mengalami kesulitan, hubungi administrator sistem untuk bantuan lebih lanjut.
