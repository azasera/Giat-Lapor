## CATATAN - TEMPLATE SIAP PAKAI

### Template 1: Supervisi Rutin

```
Supervisi Semester 1 TA 2025/2026

Mohon persiapkan:
- RPP Tahfidz
- Buku Mutaba'ah Santri
- Dokumentasi Progress Hafalan

Supervisi di kelas saat pembelajaran.
Durasi: 60-90 menit.
```

---

### Template 2: Supervisi Tematik

```
Supervisi Tematik: Peningkatan Bacaan Tajwid

Fokus evaluasi pada kualitas bacaan dan koreksi.

Siapkan:
- Contoh video pembelajaran
- Rekaman bacaan santri
- Metode koreksi yang digunakan

Konfirmasi H-3 via WhatsApp.
```

---

### Template 3: Follow-Up

```
Supervisi Follow-Up

Evaluasi progress perbaikan dari supervisi sebelumnya.

Siapkan bukti:
- Implementasi rekomendasi
- Dokumentasi perbaikan
- Kendala yang dihadapi

Akan ada sesi diskusi setelah observasi.
```

---

### Template 4: Persiapan Akreditasi

```
Supervisi Persiapan Akreditasi 2026

Dokumen wajib:
- RPP lengkap 1 semester
- Buku Mutaba'ah semua santri
- Dokumentasi pembelajaran
- Sertifikat pelatihan

Pastikan dokumen tertata rapi.
```

---

### Template 5: Supervisi Singkat

```
Supervisi Rutin Bulanan

Evaluasi umum pembelajaran.
Tidak ada dokumen khusus.
Durasi: 30-45 menit.
```

## TIPS CEPAT

### LAKUKAN:
- Isi Area Fokus jika ada tema khusus
- Isi Catatan jika ada instruksi penting
- Kosongkan jika supervisi umum
- Gunakan template untuk efisiensi

### JANGAN:
- Terlalu banyak area fokus (max 3)
- Catatan terlalu panjang
- Instruksi yang tidak jelas
- Lupa konfirmasi dengan guru
